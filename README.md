# Backend-Developer-Intern-Project-Assignment-
# Scalable REST API with Authentication & Role-Based Access

## 🚀 Project Overview
This project is a scalable backend system with JWT authentication,
role-based access control, and CRUD operations with a basic React frontend.

---

## 🛠 Tech Stack
Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcrypt
- Swagger API Docs

Frontend:
- React.js
- Axios
- React Router

---

## ✅ Features
- User Registration & Login
- Password Hashing
- JWT Authentication
- Role-Based Access (User/Admin)
- Protected Routes
- Task CRUD APIs
- API Versioning
- Error Handling Middleware
- Swagger Documentation
- Frontend Dashboard
